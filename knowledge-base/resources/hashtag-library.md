# Hashtag Library & Strategy

## Primary Brand Hashtags (Always Use)

### Core Property Tags
- **#CasaCaputMundi** - Brand hashtag principale
- **#RomaCaputMundi** - Brand + location combination

**Monthly**: Review performance, retire low-performers
**Seasonally**: Update seasonal hashtags
**Event-driven**: Add new event hashtags
**Trending**: Incorporate relevant trending tags

## Hashtag Usage Guidelines

### Optimal Number
**Facebook posts**: 5-8 hashtags maximum
**Instagram stories**: 10-15 hashtags
**Avoid**: Hashtag stuffing (looks spammy)

### Placement Strategy
**Natural integration**: Work hashtags into caption flow
**End of caption**: Group remaining hashtags at end
**Avoid**: Separate comment with only hashtags

### Format Consistency
**Capitalization**: #CasaCaputMundi (Pascal case for readability)
**Language mixing**: Italian + English OK in same post
**Spacing**: No spaces in hashtags (#Casa CaputMundi = wrong)

## Prohibited/Avoid Hashtags

### Overused Generic Tags
- **#Love** - Too generic, no targeting value
- **#Beautiful** - Meaningless for algorithm
- **#Amazing** - Overused, low relevance
- **#Perfect** - Generic, no local value

### Competitor Brand Tags
- Never use other accommodation brand hashtags
- Avoid competitor property-specific tags
- Don't hijack competitor campaigns

### Inappropriate Context
- Political hashtags (unless directly relevant)
- Controversial topics unrelated to travel
- Trending hashtags unrelated to our content

## Hashtag Calendar Integration

### Content Calendar Alignment
**Each content type** has suggested hashtag set
**Seasonal content** includes seasonal hashtags
**Event content** includes event-specific tags
**Promotional content** includes offer-related tags

### Monthly Hashtag Planning
**Week 1**: Neighborhood + local experience focus
**Week 2**: Property + comfort focus
**Week 3**: Rome events + cultural focus

This hashtag strategy ensures consistent discoverability while maintaining relevance and avoiding spam perception.
